How it work

Informational section of the site, which introduces the user to how the device works, namely the parts used and their characteristics

NodeMCU V3

NodeMCU is a developer board based on the ESP8266 chip which is a UART-WiFi module with ultra-low power consumption.

RFID RC522 +

This module can be used for various commercial applications, including access control, automatic identification, tracking of things, etc.

DISPLAY 1601

Allows you to quickly assemble a layout of a device for displaying and processing information from sensors.

TEMPERATURE SENSOR

Is used to measure the body temperature of a specific user at the entrance to the premises of the educational institution.
